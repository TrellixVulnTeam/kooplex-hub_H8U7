Help
Volumes contain installed conda environments, python and other packages.
  • Any number volume can be selected.
  • Volumes will be mounted under /vol/"volumename"
  • When you run a notebook only one conda environment can enabled at once, but all the other packages will be available like yaehmop or siesta.
Storage volumes will be mounted also under /vol/. They contain large amount of data. Only those will be visible here to which you have access.
