Help

Select a project starting template, which can be either an empty template or you can clone an already available project.

When selecting an empty template choose by the image name. The basic image provides your project with all the necessary packages.

In case you clone a project the following project groups are defined:

  • mine: those projects you created so far.
  • shared: those projects other users created but you are a collaborator of.
  • public: those projects that anyone can see.
