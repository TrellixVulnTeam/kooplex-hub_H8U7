Mount points have been renamed. The new format is the project's clean name suffixed by a project identifier. In single project container project identifier serves as user's primary group id.

In envirironments with more than one projects files are created in the users group. The creator of folders and file needs to take care of setting the proper group in order to enable collaborators to access their content. For that in a terminal newgrp prXXXXX and chgrp prXXXXX <file> are to be run, where prXXXXX stand for the project identifier.
